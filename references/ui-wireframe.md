# UI 草稿图与线框图模板

本文档提供从产品思考生成 UI 草稿图和线框图的模板和指南。

## 1. 产品思考 → UI 映射

### 1.1 问题到UI要素映射表

| 产品思考问题 | UI 要素 | 说明 |
|-------------|---------|------|
| Q1 (解决什么问题) | 页面标题、核心价值主张 | 页面主题 |
| Q2 (目标用户) | 风格定位、色彩倾向 | 面向用户群体 |
| Q3 (使用场景) | 页面布局、交互方式 | 碎片/专注/工作 |
| Q10 (角色权限) | 功能入口、权限菜单 | 显示/隐藏逻辑 |
| Q11 (前置条件) | 登录/验证状态 | 空态/登录态 |
| Q12 (次数限制) | 计数器、剩余次数显示 | 边界提示 |
| Q16 (成功样子) | 成功状态 UI | 完成页面 |
| Q17 (失败样子) | 错误提示 UI | 报错样式 |
| Q18 (边界) | 边界状态显示 | 空数据/最大值 |
| Q19 (成功反馈) | Toast/通知/跳转 | 反馈方式 |
| Q20 (响应时间) | Loading 状态 | 加载效果 |

## 3. 线框图描述模板

### 3.1 布局结构

```markdown
## 线框图描述: {页面名称}

### 布局结构
| 区域 | 高度/占比 | 内容 | 备注 |
|------|----------|------|------|
| 状态栏 | 44px | 时间/电池/信号 | iOS: 44px |
| 导航栏 | 44px | 返回/标题/操作 | 可隐藏 |
| 标签栏 | 49px | Tab图标+文字 | 可隐藏 |
| 主内容 | 自适应 | 核心功能 | 可滚动 |
| 操作栏 | 56px | 固定底部 | 可浮动 |

### 栅格系统
- 移动端: 12列, 边距16px, 间距12px
- 平板: 12列, 边距24px, 间距16px
- 桌面: 12列, 边距48px, 间距24px

### 响应式断点
| 设备 | 宽度 | 布局变化 |
|------|------|---------|
| 手机 | <768px | 单栏 |
| 平板 | 768-1024px | 双栏 |
| 桌面 | >1024px | 多栏+侧边栏 |
```

### 3.3 交互流程

```markdown
### 页面交互流程

#### 基础流程
1. 用户进入页面 → 加载默认状态
2. 用户点击操作 → 触发交互
3. 请求处理中 → 显示加载状态
4. 请求成功 → 更新UI/跳转/提示
5. 请求失败 → 显示错误提示

#### 特殊状态
| 场景 | 显示 | 操作 |
|------|------|------|
| 空数据 | 空态插图+提示文字 | 点击重试/跳转添加 |
| 网络错误 | 错误提示+重试按钮 | 点击重试 |
| 加载中 | Skeleton/Loading | 等待 |
| 权限不足 | 提示+去开通按钮 | 点击跳转 |
```

### 4.2 页面状态

```markdown
## 页面状态流

```
[加载中] → [成功] → [内容页]
    ↓
[错误] → [重试] → [加载中]
    ↓
[空数据] → [添加] → [表单页]
```

| 状态 | 条件 | 显示内容 |
|------|------|---------|
| 加载中 | 接口请求中 | Loading/Skeleton |
| 空数据 | 数据为空 | 空态插图+提示 |
| 正常 | 有数据 | 正常内容 |
| 部分数据 | 加载部分 | 内容+加载更多 |
| 错误 | 请求失败 | 错误提示+重试 |
```
