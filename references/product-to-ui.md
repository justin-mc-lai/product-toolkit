# 产品思考转换为 UI 设计指南

本文档说明如何将产品思考的结果转换为 UI 设计，包括草稿图、线框图和 UI 规范。

## 1. 核心流程

```
产品思考 (5轮追问)
    ↓
提取 UI 要素 (本指南)
    ↓
生成草稿图 (ui-wireframe.md)
    ↓
生成线框图描述 (ui-wireframe.md)
    ↓
生成 UI 规范 (ui-spec.md)
```

## 2. 产品思考 → UI 映射表

### 2.1 第一轮：澄清问题 → 页面基础

| 问题 | UI 要素 | 输出 |
|------|---------|------|
| Q1: 解决什么问题？ | 页面标题、核心价值主张 | 页面主题、功能标题 |
| Q2: 目标用户是谁？ | 用户画像 → 风格定位 | 色彩倾向、视觉风格 |
| Q3: 使用场景？ | 布局方式、交互模式 | 移动端/PC、碎片/专注 |
| Q4: 用户现在怎么解决？ | 竞品分析 → 差异化 | 改进点、创新点 |
| Q5: 为什么现在做？ | 优先级 | MVP/增强功能 |

### 2.2 第二轮：探索假设 → 功能结构

| 问题 | UI 要素 | 输出 |
|------|---------|------|
| Q6: 不做会怎样？ | 风险提示 | 降级方案、空态 |
| Q7: 假设了什么？ | 前提条件 | 前置检查、引导 |
| Q8: 最大风险？ | 边界处理 | 错误提示、容错 |
| Q9: 目标是什么？ | 核心指标 | 成功指标展示 |

### 2.3 第三轮：用户故事边界 → 界面元素

| 问题 | UI 要素 | 输出 |
|------|---------|------|
| Q10: 哪些角色可用？ | 权限菜单、功能入口 | 角色区分、权限控制 |
| Q11: 什么情况下不能用？ | 前置状态 | 登录检查、权限校验 |
| Q12: 有次数限制？ | 计数器 | 剩余次数显示 |
| Q13: 数据从哪里来？ | 数据展示 | 列表/详情/图表 |
| Q14: 需要权限？ | 权限组件 | 角色判断、隔离 |
| Q15: 可撤销？ | 撤销操作 | 撤回按钮、确认弹窗 |

### 2.4 第四轮：验收标准 → 交互反馈

| 问题 | UI 要素 | 输出 |
|------|---------|------|
| Q16: 成功是什么样子？ | 成功状态 | 完成页、成功Toast |
| Q17: 失败是什么样子？ | 错误状态 | 报错提示、重试 |
| Q18: 边界在哪里？ | 边界状态 | 最大值提示、空态 |
| Q19: 怎么知道成功了？ | 反馈方式 | Toast/跳转/通知 |
| Q20: 响应时间要求？ | 加载状态 | Loading/Skeleton |

### 2.5 第五轮：评估影响 → 后续流程

| 问题 | UI 要素 | 输出 |
|------|---------|------|
| Q21: 对其他功能的影响？ | 关联入口 | 跳转链接、推荐 |
| Q22: 怎么看效果？ | 数据埋点 | 统计展示 |
| Q23: 数据从哪里获取？ | 数据源 | 接口、埋点 |
| Q24: 回滚方案？ | 回退操作 | 撤销、恢复 |

## 3. 转换示例

### 3.1 示例：电商收藏功能

**产品思考结果**:
- Q1: 解决用户收藏商品的需求
- Q2: 消费者、购物爱好者
- Q3: 碎片时间、购物场景
- Q10: 所有登录用户
- Q11: 未登录引导登录
- Q12: 无限制
- Q16: 显示收藏成功Toast，跳转收藏列表
- Q17: 显示错误提示
- Q19: Toast + 跳转

**转换后的 UI**:

```markdown
## 草稿图: 收藏页面

┌─────────────────────────────────────────┐
│  [<]  我的收藏 (3)          [编辑]      │
├─────────────────────────────────────────┤
│  [全选]                        [删除]  │
├─────────────────────────────────────────┤
│  ┌─────┐                               │
│  │     │ 商品名称商品名称商品名称        │
│  │ 图片 │ ¥99.00           [已收藏]  │
│  │     │ 店铺名称                     │
│  └─────┘                               │
│  ┌─────┐                               │
│  │     │ 商品名称商品名称商品名称        │
│  │ 图片 │ ¥199.00          [已收藏]  │
│  │     │ 店铺名称                     │
│  └─────┘                               │
│                ...                      │
├─────────────────────────────────────────┤
│  [tab:全部][tab:降价][tab:有货]        │
└─────────────────────────────────────────┘

### 状态映射
| 场景 | UI |
|------|-----|
| 初始 | 全部商品列表 |
| 空数据 | 空态插图+去逛逛按钮 |
| 加载中 | Skeleton |
| 收藏成功 | Toast: 收藏成功 |
| 取消收藏 | Toast: 已取消收藏 |
| 网络错误 | 错误提示+重试按钮 |
```

## 4. 快速转换清单

### 4.1 必选元素

- [ ] 页面标题 (来自 Q1)

### 4.2 可选元素

## 5. 使用子命令

### 5.1 生成草稿图

```bash
/product-toolkit:wireframe 登录页面
/product-toolkit:wireframe 电商商品详情页
```

### 5.2 生成 UI 规范

```bash
/product-toolkit:ui-spec 电商详情页
/product-toolkit:ui-spec 登录页面
```

### 5.3 完整流程

```bash
# 1. 产品思考
/product-toolkit:think 我想做社区点赞功能

# 2. 生成草稿图
/product-toolkit:wireframe 社区点赞

# 3. 生成 UI 规范
/product-toolkit:ui-spec 点赞功能
```
