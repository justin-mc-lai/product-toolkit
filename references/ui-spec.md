# UI 设计规范

本文档提供 UI 设计规范的完整模板，包括颜色、字体、间距、组件等。

## 6. 动画规范

### 6.1 时长

```markdown
### 动画时长
| 名称 | 时长 | 用途 |
|------|------|------|
| 立即 | 0ms | 无动画 |
| 快速 | 100ms | 颜色变化、hover |
| 正常 | 200ms | 淡入淡出、位移 |
| 慢速 | 300ms | 弹窗、抽屉 |
| 更慢 | 500ms | 页面切换 |
```

### 6.2 缓动函数

```markdown
### 常用缓动
| 名称 | 函数 | 用途 |
|------|------|------|
| ease | cubic-bezier(0.4, 0, 0.2, 1) | 默认 |
| easeIn | cubic-bezier(0.4, 0, 1, 1) | 淡入 |
| easeOut | cubic-bezier(0, 0, 0.2, 1) | 淡出 |
| easeInOut | cubic-bezier(0.4, 0, 0.2, 1) | 淡入淡出 |
| spring | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 弹性 |
```
