# 验收标准模板库

## 1. 5维度基础模板（必选）

所有用户故事必须包含以下5个维度：

1. **正向流程** - 核心功能可正常使用
2. **边界校验** - 边界值/异常输入验证
3. **错误处理** - 异常场景友好提示
4. **成功反馈** - 操作后明确反馈
5. **性能** - 响应时间/并发要求

---

## 2. 用户故事 → QA 用例完整流程

- [ ] 正向流程: 点击收藏按钮，收藏成功，按钮变为已收藏
